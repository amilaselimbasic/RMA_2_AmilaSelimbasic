**Gdje danas** – Android aplikacija

Aplikacija **Gdje danas** je Android mobilna aplikacija razvijena u Android Studio okruženju koristeći Java programski jezik. Cilj aplikacije je prikaz i pregled različitih događaja (koncerti, predstave, sportski događaji i festivali) u BiH, uz mogućnost spremanja favorita i liste „želim posjetiti“.

Funkcionalnosti

- Prikaz liste događaja (RecyclerView)
- Detaljan prikaz svakog događaja
- Pretraga događaja po nazivu
- Filtriranje po kategorijama (muzika, sport, predstave, ostalo)
- Dodavanje događaja u:
  - Favorite
  - Želim posjetiti
- Prikaz sačuvanih lista u profilu korisnika
- Firebase autentifikacija:
  - Registracija korisnika
  - Prijava
  - Reset lozinke
  - Logout
- Dark mode (noćni režim)
- Bottom navigation između ekrana

Tehnologije

- Java (Android)
- Android Studio
- RecyclerView + Adapter
- SharedPreferences + Gson
- Firebase Authentication
- Firebase Realtime Database
- Material Design
